🔧 Configuración de Google Apps Script

  1. Creá una nueva hoja de cálculo en Google Sheets
  2. Andá a Extensiones → Apps Script
  3. Borrá todo y pegá el código del script (ver abajo)
  4. Guardá y hacé click en Implementar → Nueva implementación
  5. Elegí tipo: Aplicación web
  6. Ejecutar como: Yo
  7. Acceso: Cualquier persona
  8. Copiá la URL de la Web App
  9. Pegala en GOOGLE_SCRIPT_URL en este archivo
